ASTRONEER: FORCE-LAND SHUTTLE

This is a fix for an Astroneer bug where a shuttle gets stuck in orbit and you can't bring it back. It happens in multiplayer/co-op after a lag spike or a player disconnecting while in orbit.
The fix repositions the shuttle onto the pad then clears lost/orbit state to landed state and sets ExitSuppressionCount = 0 on its seats.
After landing, you can download the new RECOVERED_*.savegame, then copy it into your SaveGames folder.
⚠ Read this before using
  • BACK UP your save file first.
  • Do NOT overwrite your original save with the RECOVERED file - keep both.
  • This was tested only with my own save files and my own specific bug.
  • No guarantee it will work 100% for your save - use at your own risk.
